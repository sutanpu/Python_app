# 💱 為替レート可視化アプリ

通貨ペアを選ぶと、過去30日間の為替推移グラフ、最新レート、CSV/PDFレポートが表示されるWebアプリです。

## 🔧 主な機能

- 通貨ペアの選択（USD, EUR, JPY, GBP）
- 過去の為替レート推移グラフ表示
- 最新レートと変動値の表示
- CSV / PDFレポートの出力
- 日本語PDFレポート対応
- 経済ニュース（関連API）表示← **準備中**
